# Springload Form Validation


## Setup

If you want to run this repo locally


Clone the repo:
```
git clone https://github.com/magda-gierg/form_validation
cd form_validation
```

Install the Modules:
```
npm instal or yarn install
```

Run the App:
```
npm start or yarn start
```
